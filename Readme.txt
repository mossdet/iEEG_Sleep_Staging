SleepSEEG: automatic sleep scoring using intracranial EEG recordings (Frauscher 2022)
paper: https://iopscience.iop.org/article/10.1088/1741-2552/ac6829
code: https://zenodo.org/records/6412063


SleepSEEG performs automatic sleep scoring of intracranial EEG files in Matlab, as described in 
'von Ellenrieder et al., SleepSEEG: Automatic sleep scoring using intracranial EEG recordings only. Journal of Neural Engineering, 2022'.

It requires a complete night of iEEG data, in EDF or EDF+C format. The file SleepSEEG_models.mat must be in the same folder as SleepSEEG.m. 
For information on how to use this function type 'help SleepSEEG' in Matlab.
